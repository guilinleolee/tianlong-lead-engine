### 优质项目识别标准

#### 项目类型优先级
**优先接**：
- ✅ 性能优化（单价高，周期短）
- ✅ 架构升级（单价高，周期适中）
- ✅ 故障排查（紧急，单价高）
- ✅ 技术评估（单价低，转化率高）

**谨慎接**：
- ⚠️ 从0开发新项目（周期长，需求不稳定）
- ⚠️ 维护型项目（单价低，价值低）

**不接**：
- ❌ 需求不明确的项目
- ❌ 预算过低的项目（低于5000）
- ❌ 客户沟通困难的项目

### 私信转化话术

#### 话术1：技术问题咨询 → 付费评估
**用户**：老师，我遇到了数据库性能问题...
**回复**：
```
您好！从您描述的情况来看，这个问题我之前处理过类似案例。
数据库慢查询优化需要先看具体情况，我这边可以帮您做个免费
的初步评估。

评估内容：
1. 分析慢查询日志，定位问题SQL
2. 检查索引使用情况
3. 给出优化建议

评估流程：
- 您提供数据库慢查询日志
- 我分析后给您出评估报告
- 报告里会有详细优化方案

如果需要我帮您实施优化，再谈合作细节。
您看方便把日志发我看看吗？
```

#### 话术2：项目需求咨询 → 成交
**用户**：我们有个项目想做...
**回复**：
```
您好！这个项目方向不错。从技术角度给您一些建议：

**技术方案评估**：
- MVP阶段建议单体架构，后期再微服务
- 预算充足的话可以用云服务，节省运维成本
- 用户数据要注意合规，提前设计脱敏方案

**我的服务**：
我这边可以提供全程技术服务，包括：
1. 技术方案设计（含架构图+技术选型）
2. 核心功能开发（按模块分阶段交付）
3. 上线部署+运维支持

**服务流程**：
- 第1步：技术评估（免费，1-2天）
- 第2步：方案设计（付费，3-5天）
- 第3步：开发实施（按阶段付费）

**收费标准**：
- 技术评估：免费
- 方案设计：3000-5000
- 开发实施：根据需求评估，一般1w起步

方便的话，可以先约个语音沟通下需求？
```

#### 话术3：长期合作 → 顾问服务
**用户**：我们公司长期需要技术支持...
**回复**：
```
您好！长期技术合作这块我这边有几种合作模式：

**模式1：项目制**
- 适合：具体项目开发
- 收费：按项目报价，一般1w起
- 交付：分阶段，每阶段验收后付款

**模式2：顾问制**
- 适合：团队技术指导+架构评审
- 收费：5000-10000/月
- 服务：
  * 每周1次技术评审会议
  * 架构方案review
  * 关键技术问题咨询
  * 故障应急支持

**模式3：混合制**
- 适合：既有项目又有长期需求
- 收费：项目制 + 顾问制组合
- 优势：长期合作有优惠

建议可以先从小项目合作试试，看是否合适。
您这边具体是哪种需求呢？
```
